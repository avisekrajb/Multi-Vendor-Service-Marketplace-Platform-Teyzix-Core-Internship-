# TEYZIX - Nepal's #1 Freelance Service Marketplace

![TEYZIX Logo](https://via.placeholder.com/1200x400?text=TEYZIX+Marketplace)

## 🚀 Overview

TEYZIX is a comprehensive freelance service marketplace platform connecting customers with skilled service providers in Nepal. Built with the MERN stack (MongoDB, Express.js, React.js, Node.js), it offers a seamless experience for both customers looking for services and providers offering their expertise.

## ✨ Features

### For Customers
- 🔍 Browse and search services with advanced filters
- 📝 Submit service requests with custom requirements
- 💬 Real-time chat with providers
- 📊 Track project progress
- ⭐ Leave reviews and ratings
- 💰 Secure payments in NPR
- 📱 Fully responsive design

### For Providers
- 🛍️ Create and manage service listings
- 📋 Accept and manage project requests
- 💬 Real-time customer communication
- 📈 Track earnings and performance
- ⭐ Respond to customer reviews
- 👤 Complete professional profile

### For Admins
- 📊 Platform analytics dashboard
- 👥 User management (ban/unban)
- 🛍️ Service approval system
- 📋 Project monitoring
- ⚖️ Dispute resolution
- 📄 Report generation (CSV/Excel)
- 📝 Activity logging

### Technical Features
- 🔐 JWT Authentication with Refresh Tokens
- 📧 Email notifications (Welcome, Status Updates, Reviews)
- 🖼️ Cloudinary image upload
- 💬 WebSocket real-time chat
- 🔄 Real-time online status
- 🌙 Dark mode support
- 📱 Fully responsive design
- 🐳 Docker support

## 🛠️ Tech Stack

### Frontend
- React.js 18
- Tailwind CSS
- React Router v6
- Socket.io Client
- Axios
- React Hot Toast
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- Socket.io
- JWT
- Nodemailer
- Cloudinary
- Multer

## 📋 Prerequisites

- Node.js (v18 or higher)
- MongoDB Atlas account or local MongoDB
- Cloudinary account
- Gmail account (for email notifications)
